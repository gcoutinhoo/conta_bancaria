# 💰 Conta Bancária

## 📌 Sobre o projeto
Projeto desenvolvido para simular operações bancárias básicas, como criação de contas, depósitos, saques e consultas de saldo.

O objetivo é aplicar conceitos de Programação Orientada a Objetos (POO), lógica de programação e boas práticas no desenvolvimento com JavaScript/TypeScript.

---

## 🛠️ Tecnologias utilizadas
- JavaScript / TypeScript
- Node.js

---

## ⚙️ Funcionalidades
- ✅ Criação de conta bancária  
- ✅ Depósito de valores  
- ✅ Saque de valores  
- ✅ Consulta de saldo  
- ✅ Validações básicas de operações  

---

## 🚀 Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/gcoutinhoo/conta_bancaria.git

# entrar na pasta
cd conta_bancaria

# instalar dependências (se houver)
npm install

# executar o projeto
npm start
